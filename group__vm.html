Detailed Description

Allows user applications to execute safely in an abstract environment that's consistent in differing hardware environments.

At its base level, the VM has a simple Virtual Machine Language that provides program flow control and basic functionality. It also provides a VM_PROC::stack on which to store data that is being worked with.

More advanced operating system functionality (e.g. file and screen access) is handled by Virtual Machine Syscalls and the SYSC operation.
